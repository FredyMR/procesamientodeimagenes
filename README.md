# procesamientodeimagenes
Proyecto de la Materia de Procesamiento de Imagenes, En el cual se aplican diversos calculos matriciales a imagenes. Por ejemplo: 

*Escala de Grises

*Escala Negativo

*Operaciones Logicas

*Umbral y Umbral Inverso

*Operaciones Con Numeros

*Histograma

*Strecht

*Compresión

*Convolución

*Rotación

*Efecto Espejo
